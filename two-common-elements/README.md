# Two Common Elements

Find the common integers in two given sorted arrays of integers if they exists. Return an sorted array the common integers or an empty array.

## **Examples:**

```javascript
Input: arrayA = [1, 2, 5, 6, 10];
arrayB = [4, 5, 6, 8, 9];
Output: [5, 6];

Input: arrayA = [1, 2, 3, 4, 5];
arrayB = [6, 7, 8, 9];
Output: [];
```
